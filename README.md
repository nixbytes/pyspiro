# Basic build of wheel

```
python3 setup.py bdist_wheel
```

# Install wheel

```
python3 -m pip install pyspiro-2021.2.4-py3-none-any.whl
```

# How to run pyspiro

For a random images generation

```python
pyspiro

```

```python
pyspiro --sparams 300 100 0.9

```
